Functional tests

Functional tests are used to simulate user interactions with your page. Here's how to get started.

  1. Create a file like 'widget_test.js' in the 'test/functional' directory and add the following code: new jQuery.Test.Functional('widget',{ test_truth: function() { this.assert(true); } });
  2. Create a file called 'YOURAPPNAME_test.js' in your 'app' directory and add the following code: include.functional('widget')
  3. Reload your page.

For more information, check out the documentation.
